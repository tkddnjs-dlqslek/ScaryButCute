# Chrome Web Store Description

## Name
ScaryButCute - Watch scary videos comfortably

## Short Description (132 chars max)
Place cute animals around horror/thriller videos for visual comfort. Supports YouTube, Netflix, Disney+.

## Detailed Description (Korean)

### 무서운 영상도 편안하게, ScaryButCute

공포/스릴러 콘텐츠를 보고 싶지만 무서워서 못 보시나요?
ScaryButCute는 영상 주변을 귀여운 동물 친구들로 채워 안정감을 줍니다.

**지원 플랫폼**
- YouTube
- Netflix
- Disney+

**8종 동물 테마**
- 강아지, 고양이, 여우, 카피바라, 토끼, 레서판다, 페럿, 눈표범
- 동물 친구들 믹스 모드

**주요 기능**
- Alt+C 단축키로 빠르게 켜고 끄기
- 영상 크기 조절 (30%~90%)
- 매번 랜덤 배치로 새로운 느낌
- 네트워크 요청 없음 - 모든 이미지 내장
- 개인정보 수집 없음

**사용법**
1. 영상 재생 페이지에서 확장프로그램 아이콘 클릭
2. 컴포트 모드 켜기
3. 원하는 동물 선택
4. 편안하게 시청!

## Detailed Description (English)

### Watch scary videos comfortably with ScaryButCute

Want to watch horror/thriller content but too scared?
ScaryButCute fills the area around the video with cute animal friends for visual comfort.

**Supported Platforms**
- YouTube
- Netflix
- Disney+

**8 Animal Themes**
- Dog, Cat, Fox, Capybara, Rabbit, Red Panda, Ferret, Snow Leopard
- Animal friends mix mode

**Key Features**
- Alt+C shortcut for quick toggle
- Video size adjustment (30%~90%)
- Random layout every time
- No network requests - all images bundled
- No personal data collection

**How to Use**
1. Go to a video page and click the extension icon
2. Turn on Comfort Mode
3. Choose your animal friend
4. Watch comfortably!
